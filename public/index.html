This silly website can help you make mashups of content you find online.

Controls

Use the menu to add new content onto the page, or even edit the CSS / JavaScript if you're an advanced user.

Once you've added something to the page, you can click and drag to move it around, or grab the edges to rotate and resize it. Double clicking on an object will open an overlay to edit its options.

To remove something from the page, right click on it. If you hold the "control" key while right clicking, the element will explode.

Shortcuts

There are also a number of keyboard shortcuts you can use to make editing easier:

  • Escape - Open and close the menu
